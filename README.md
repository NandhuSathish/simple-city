# Cute Fantasy City Builder

A web-based 2D city builder built with [Phaser 4](https://phaser.io) and TypeScript, using the [Cute Fantasy RPG](https://kenmi-art.itch.io/cute-fantasy-rpg) asset pack by kenmi-art. Place buildings, manage resources, grow crops, and watch villagers bring your town to life — all in the browser.

**Assets:** The Cute Fantasy Premium sprite pack is required at `G:\Cute_Fantasy\` (not committed — Premium licence forbids redistribution). After cloning, copy `grass_tile.png` and `player.png` into `public/assets/` or run the atlas pipeline.

## Dev commands

```bash
npm install          # install dependencies
npm run dev          # Vite dev server with HMR → http://localhost:5173
npm run build        # production build to dist/
npm run preview      # preview the production build locally
```
